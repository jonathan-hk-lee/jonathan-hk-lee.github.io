Sticky Bun is a Jello Simulator based on the Finite Element Method. Uses TetGen to tetrahedralize meshes and Partio to write .bgeo files for Houdini. All scenes were rendered in Houdini using Mantra.

Focused on collisions using signed distance fields, general debugging, and creating testcases for interesting scenes.
